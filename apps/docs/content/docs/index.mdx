Unofficial TypeScript tools for [Suno](https://suno.com) — AI music generation:

- **`@yabbal/suno-sdk`** — TypeScript SDK to integrate Suno into your applications
- **`@yabbal/suno-cli`** — CLI to generate music, manage songs, lyrics, and more from the terminal

## Installation

<Tabs items={["SDK", "CLI"]}>
  <Tab value="SDK">
    ```bash title="Terminal"
    npm install @yabbal/suno-sdk
    ```
  </Tab>
  <Tab value="CLI">
    <Tabs items={["npm", "Homebrew", "Source"]}>
      <Tab value="npm">
        ```bash title="Terminal"
        npm install -g @yabbal/suno-cli
        ```
      </Tab>
      <Tab value="Homebrew">
        ```bash title="Terminal"
        brew tap yabbal/tap && brew install suno
        ```
      </Tab>
      <Tab value="Source">
        ```bash title="Terminal"
        git clone https://github.com/yabbal/suno.git
        cd suno && pnpm install && pnpm build
        pnpm link --global
        ```
      </Tab>
    </Tabs>
  </Tab>
</Tabs>

## Authentication

Suno has no public API. Authentication uses an **access token** extracted from your browser session.

**How to get your token:**

1. Open [suno.com](https://suno.com) in your browser (logged in)
2. Open DevTools (F12) → **Network** tab
3. Find any request to `studio-api.prod.suno.com`
4. Copy the `Authorization` header value (the JWT after `Bearer `)

```bash title="Terminal"
# Option 1: interactive login (guides you through the steps)
suno auth login

# Option 2: environment variable
export SUNO_COOKIE="eyJhbG..."
```

## Quick Start

<Tabs items={["SDK", "CLI"]}>
  <Tab value="SDK">
    <Steps>
      <Step>
        ### Configure authentication

        ```bash title="Terminal"
        export SUNO_COOKIE="your-suno-cookie-value"
        ```
      </Step>

      <Step>
        ### Use the SDK

        ```ts title="app.ts"
        import { SunoClient } from "@yabbal/suno-sdk";

        const client = new SunoClient({ cookie: process.env.SUNO_COOKIE });

        // Generate a song
        const songs = await client.songs.generate({ prompt: "A happy jazz song about coding" });

        // List your songs
        const mySongs = await client.songs.list();
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab value="CLI">
    <Steps>
      <Step>
        ### Log in

        ```bash title="Terminal"
        suno auth login
        ```
      </Step>

      <Step>
        ### Generate a song

        ```bash title="Terminal"
        suno songs generate --prompt "A happy jazz song about coding"
        ```
      </Step>

      <Step>
        ### List your songs

        ```bash title="Terminal"
        suno songs list --format table
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
